# Integration Test Vue 2

Vue 2 + Element UI 集成测试项目，用于测试 @form-renderer/preset-element-ui 预设包。

## 安装

```bash
pnpm install
```

## 运行

```bash
pnpm serve
```

## 项目结构

```
src/
├── main.js           # 入口文件
├── App.vue           # 主应用
└── examples/         # 测试用例
    ├── 01-basic-form.vue       # 基础表单
    ├── 02-all-widgets.vue      # 所有组件
    ├── 03-complex-form.vue     # 复杂表单
    ├── 04-dynamic-list.vue     # 动态列表
    └── 05-validation.vue       # 表单校验
```
